## 🚀 GUIA DE USO

### 1. Instalar
```bash
cd kindred-connect
npm install
npm run dev
```

### 2. Acessar
```
http://localhost:3000
```

### 3. Cadastrar
- Clique "Começar Grátis"
- Preencha formulário
- Código: `ZAPMRO2026`
- Clique "Criar Conta"

### 4. Explorar
- **Dashboard** - Stats e atividades
- **Chats** - 4 conversas de demo
- **CRM** - Kanban com 3 colunas
- **IA Agente** - Config de assistente
- **Fluxos** - Criar automações
- **Eventos** - Agendar compromissos
- **Configurações** - Perfil e notificações
